# MNIST TensorFlow Demo (1-click via GitHub Codespaces)

Deze repo bevat:
- Een **1-click demo** (Gradio) om zelf cijfers te tekenen en te laten voorspellen.
- Jupyter notebooks voor training/experimenten.

> Eerste run kan even duren: MNIST wordt gedownload en het model kan (eenmalig) trainen.

---

## 1) 1-click demo (GitHub Codespaces)

1. Open deze repo op GitHub.
2. Klik **Code → Codespaces → Create codespace on main**.
3. Wacht tot de container klaar is (requirements worden automatisch geïnstalleerd).
4. Start de demo:

```bash
python app/gradio_app.py
